## HandConfirmation 仕様メモ

- 役割: 2 枚のホールカードを表示し、スマホの左下から右方向のスワイプで裏面から表面へフリップする UI。
- タッチ制約: `pointerType === "touch"` のみ処理。完了後はロックして二度と動かない。
- スワイプ開始条件: コンテナ左下のガイド領域内で指を置く。領域はコンテナの 35%×35% で、`absolute bottom-0 left-0` に配置される (ガイド自体は遅延表示)。
- 進捗計算: 横方向の移動距離 / (コンテナ幅の 60%) を 0〜1 にクランプ。1.0 到達時に即ロック。
- 非線形速度: 進捗 0〜0.25 までは表示側を 2 倍速で回転させ、0.25 以降は線形で 1.0 へつなぐ。
- 離脱時の自動判定: タッチ終了時にロックされていなければ、進捗が 3/5 以上で自動で 1.0 にスナップしてロック、それ未満なら 0 に戻す。
- ガイド表示: hands がレンダリングされた後、3 秒間スワイプ操作がなければ「Swipe start」のガイド枠を表示。タッチ操作が起きるとタイマーをリセットしガイドを非表示にする。
